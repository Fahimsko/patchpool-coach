How it works

Three steps from patch day to ranked confidence.

1

Connect your account

Enter your Riot ID. PatchPool Coach fetches your recent ranked solo/duo match history using the official Riot API.

2

Analyse your pool

Win rate, KDA, games played, and champion mastery are aggregated per champion across your last 10–50 ranked games.

3

Get your recommendations

Each champion receives a personal impact score and a clear recommendation, updated every patch.

4

Patch alerts

Enable notifications and get a Chrome alert the moment a new patch drops that buffs or nerfs a champion in your pool.

Recommendation categories

Every champion in your pool gets one of five verdicts.

Keep Playing

Strong personal stats and no harmful patch changes. Your safest ranked pick.

Practice More

Solid experience but win rate has room to grow. Ranked-ready with more focused reps.

Risky This Patch

A significant nerf or borderline stats make this a volatile ranked pick right now.

Drop For Now

Underperforming stats and no patch buff to rescue it. Bench it this patch.

Not Enough Data

Fewer than 3 ranked games on record. Build a sample in normals first.

Riot API usage

PatchPool Coach uses only official, documented Riot API endpoints.

Endpoint Purpose
Account v1 Resolve Riot ID (game name + tagline) to a PUUID
Match v5 Fetch ranked solo/duo match history (queue 420) and per-game stats
Champion Mastery v4 Fetch mastery points and mastery level per champion
Data Dragon (CDN) Fetch champion metadata and current game version

🔒

API key security

All Riot API calls are made exclusively from a server-side backend proxy. The API key is stored only on the server and is never transmitted to or stored inside the browser extension. Users' match data is fetched on demand and never persisted on our servers.
